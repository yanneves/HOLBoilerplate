Build Steps:

build/HOLBuild.bat now handles this!

  1. Preprocess .less files into .css
  2. Replace calls to .less files with call to .css
  3. Run Ant Build Script
  4. Run Require (r.js) Compiler
  5. Deploy http/ for production
