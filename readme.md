# HOLBoilerplate
__H5BP-OOCSS-LESS-BBBP Boilerplate__


## Build Steps:

1. __Preprocess__ `.less` files into `.css`
2. __Move__ production files into `http/`
3. __Run__ Ant Build Script
4. __Deploy__ `http/` for production
